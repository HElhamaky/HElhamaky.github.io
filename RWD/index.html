WHAT WILL YOU LEARN?

LEVEL1 (25 HOURS)

Intro to HTML&CSS ( Making Web Pages )

You will learn how to use HTML and CSS to make web pages. HTML is markup language that surround content with, to tell browser about headings, lists, tables, etc. CSS is the style sheet language that you style the page with, to tell browsers to change the color, font, layout, and more.

LEVEL 2 (25 HOURS)

Advanced to HTML&CSS ( Responsive Web Design 1 )

In this course you will learn the fundamentals of HTML5 semantic layout and CSS3 fundamentals, how to build high quality Website and UI projects using Flexbox. Learn how to build responsive, beautiful web applications faster using less code and best practices.

LEVEL 3 (25 HOURS)

Advanced to HTML&CSS ( Responsive Web Design 2 )

In this course, we will start by a review of floats for context, but quickly moving into Flexbox and CSS grids, master CSS Grid and Flexbox, the latest tools and tricks to layout beautiful, responsive web applications with less code.
